# eCalendar Home Assistant Add-on

Daily planner add-on for Home Assistant with events, chores, to-do lists, weather, and calendar sync.

## Features

- **Calendar** — Events with month view
- **Chores** — Allocate and track household chores
- **Lists** — Multiple to-do lists with items
- **Weather** — Location-based (Open-Meteo)
- **CalDAV** — Import from iCloud, Nextcloud

## Installation

1. In Home Assistant: **Settings** → **Add-ons** → **Add-on store**
2. Click the **⋮** (three dots) → **Repositories**
3. Add: `https://github.com/matt-gun/ecalendar-ha`
4. Install **eCalendar** add-on
5. Start the add-on
6. Open **eCalendar** from the sidebar

## Requirements

- Home Assistant OS or Home Assistant Supervised
- Ingress support (built-in)

## Data

Data is stored in the add-on persistent storage at `/data`.
